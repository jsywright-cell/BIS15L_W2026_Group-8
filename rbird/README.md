# BIS15L_W2026_Group-8

Our project was on the birdwatching statistics in Yolo County from 2018-2023, and the species statistics from 2019-2021.

eBird is a program/website/app created by Cornell Lab of Ornithology in order to track species of birds across the world.

After requesting access to their data and being approved, I was able to receive the data at the following location.
https://ebird.org/data/download?p=ebd_US-CA-113_201801_202312_smp_relJan-2026.zip

Unfortunately the file was too large for github to publish the species data even compressed, so we had to remove the species data from 2018, 2022, and 2023.
